Analysis:
The error is a PDDL semantic error in the predicate usage in the PDDL problem definition, specifically in the line (at knife robot2). According to the domain file, the predicate (at ?obj - object ?loc - location) expects the second argument to be of type location, but here robot2 (type robot) is provided. Objects carried by a robot are not "at" a robot using this predicate; instead, the (carry ?robot ?obj) predicate describes this relation.

Correction:
In the scene graph and PDDL problem, do not use (at knife robot2). Instead, just use (carry robot2 knife).
