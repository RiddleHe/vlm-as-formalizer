```pddl

    (:domain blocksworld)
    (:objects
        yellow orange green purple blue red - block
        robot1 - robot
    )
    (:init
        (ontable yellow)
        (ontable orange)
        (ontable green)
        (ontable purple)
        (ontable blue)
        (clear yellow)
        (clear orange)
        (clear green)
        (clear purple)
        (clear blue)
        (handempty robot1)
    )
    (:goal (and
        (on yellow blue)
        (on red orange)
        (on orange green)
        (on green purple)
    ))
)
```
