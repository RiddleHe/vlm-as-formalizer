To generate the PDDL problem based on the given instruction and domain, we need to define the objects, their initial states, and the goal specifications.

### Step-by-Step Analysis:

1. **Objects**:
   - We have 6 blocks: orange, red, yellow, green, blue, and purple.
   - There is one robot arm named `robot1`.

2. **Initial State**:
   - The robot arm `robot1` is on the table and empty-handed (`handempty`).
   - The blocks are stacked as follows: orange over red over yellow over green over blue over purple.

3. **Goal**:
   - Create three stacks:
     - Green over Blue
     - Purple over Orange
     - Red over Yellow

### PDDL Problem Definition:

```pddl
(define (problem blocksworld_stack)
  (:domain blocksworld)
  (:objects
    orange - block
    red - block
    yellow - block
    green - block
    blue - block
    purple - block
    robot1 - robot
    table - table
  )
  (:init
    (clear orange)
    (clear red)
    (clear yellow)
    (clear green)
    (clear blue)
    (clear purple)
    (ontable orange)
    (ontable red)
    (ontable yellow)
    (ontable green)
    (ontable blue)
    (ontable purple)
    (handempty robot1)
    (on orange red)
    (on red yellow)
    (on yellow green)
    (on green blue)
    (on blue purple)
  )
  (:goal
    (and
      (on green blue)
      (on purple orange)
      (on red yellow)
    )
  )
)
```

### Explanation:

- **Objects**: We define six blocks (`orange`, `red`, `yellow`, `green`, `blue`, `purple`) and one robot (`robot1`), which is initially on the table and empty-handed.
- **Init State**:
  - Each block is clear (`clear` predicate).
  - Each block is on the table (`ontable` predicate).
  - The robot arm `robot1` is empty-handed (`handempty` predicate).
  - The blocks are stacked as specified: orange over red over yellow over green over blue over purple.
- **Goal**:
  - We want to achieve three specific stacks:
    - Green over Blue
    - Purple over Orange
    - Red over Yellow

This PDDL problem definition adheres to the constraints of the `blocksworld` domain and captures the initial state and the desired goal as per the instruction.
