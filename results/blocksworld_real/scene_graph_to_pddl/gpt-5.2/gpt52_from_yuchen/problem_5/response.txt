Scene graph:
block: red_block (red cube), yellow_block (yellow cube), orange_block (orange cube), purple_block (purple cube), blue_block (blue cube), green_block (green cube)
robot: robot_arm

ontable (block) block x is on table, not on another block:
- (ontable yellow_block)
- (ontable orange_block)
- (ontable purple_block)
- (ontable blue_block)
- (ontable green_block)

clear (block) there is no block on top of block x:
- (clear red_block)
- (clear orange_block)
- (clear purple_block)
- (clear blue_block)
- (clear green_block)

handempty (robot) if robot is not seen in the image, it can be assumed to exist and is handempty:
- (handempty robot_arm)

handfull (robot) :
- (none)

holding (block) block x is being held by the robot:
- (none)

on (block, block) block x is on block y:
- (on red_block yellow_block)

PDDL problem: (define (problem blocksworld_stacks_3)
  (:domain blocksworld)
  (:objects
    red_block yellow_block orange_block purple_block blue_block green_block - block
    robot_arm - robot
  )
  (:init
    ;; blocks on table
    (ontable yellow_block)
    (ontable orange_block)
    (ontable purple_block)
    (ontable blue_block)
    (ontable green_block)

    ;; stacking relation in the scene
    (on red_block yellow_block)

    ;; clear blocks (nothing on top)
    (clear red_block)
    (clear orange_block)
    (clear purple_block)
    (clear blue_block)
    (clear green_block)

    ;; robot state
    (handempty robot_arm)
  )
  (:goal
    (and
      (on yellow_block purple_block)
      (on blue_block red_block)
      (on orange_block green_block)
    )
  )
)
