```pddl
(define (problem blockstack)
    (:domain blocksworld)
    (:objects
        robot1 - robot
        blue - block
        orange - block
        red - block
        purple - block
        green - block
        yellow - block
    )

        (on blue orange)
        (on orange red)
        (on red purple)
        (on purple green)
        (ontable green)
        (clear yellow)
        (ontable yellow)
        (handempty robot1)
    )
    (:goal
        (and
            (on blue orange)
            (on orange red)
            (on red purple)
            (on purple green)
            (on green yellow)
            (clear blue)
            (clear yellow)
        )
    )
)
```

- The initial state (`:init`) describes the current state of the blocks and robot:

- The goal (`:goal`) specifies that `blue` should be on top of `orange`, `orange` on `red`, `red` on `purple`, `purple` on `green`, and `green` on `yellow`, with `blue` and `yellow` clear, forming the stack `blue over orange over red over purple over green over yellow`.
