; Analysis of error:
; The problem from last time stated (knife_free) in the init while the knife was being held.
; According to the domain, if the knife is held, knife_free should be false.
; Also, the bowl should be defined as type bowl not object.
; The initial state should reflect that knife is held by a robot and knife_free is not true.

(define (problem cooking_problem2)
(:domain cooking)
(:objects 
    robot1 - robot
    robot2 - robot
    cucumber - vegetable
    tomato - vegetable
    knife - tool
    bowl - bowl
    table - location
)
(:init
    (at cucumber table)
    (at tomato table)
    (at bowl table)
    (is_tool knife)
    (is_knife knife)
    (holding robot1 knife)
)
(:goal (and
    (sliced cucumber)
    (sliced tomato)
    (in_bowl cucumber bowl)
    (in_bowl tomato bowl)
))
)
